Capability
Long-horizon state management

Each Partner Org is a long-running, cross-week task object. Structured storage + Claude Memory tools persist milestones and artifacts so we don't shove all history into the context window.

Capability
Prompt Caching

The Partner Handbook, certification outline, and compliance terms are stable and large. Putting them under cache_control yields 90%+ cache hit rates and dramatically reduces cost.

Capability
Human-in-the-loop

Submitting applications, signing agreements, and external comms all require Champion approval. Card-style follow-ups, no silent auto-approve.
