能力点
长程状态管理

每个 Partner Org 是一个跨周的长任务对象,用结构化存储 + Claude Memory 工具保存 milestones 与 artifacts,避免把所有历史塞进上下文。

能力点
Prompt Caching

Partner Handbook / 认证大纲 / 合规条款稳定且体量大,放进 cache_control 后命中率可达 90%+,显著降低成本。

能力点
人在环

提交申请、签署协议、对外通讯强制经 Champion 批准。卡片式追问,不设默认放行。
